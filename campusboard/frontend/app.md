# CampusBoard App — Student Bulletin Board

> The main listing feed for verified university students.

**URL:** https://campusboard.app/app
**Login required:** Yes (university email, .edu.tr)

---

## Overview

The CampusBoard app is the main interface where students browse, search, filter, and manage listings. All listings are from verified university students at the same institution.

---

## Browsing and Filtering

- **Search:** Full-text search across listing titles and descriptions
- **Category filter:** Ders Notu, Staj, Yurt/Kiralık Oda, İkinci El, Etkinlik, Genel
- **Faculty filter:** Filter by university faculty/department
- **Status filter:** Active listings only, or include expired ones

---

## Listing Cards

Each listing card shows:
- Category badge
- Title
- Short description excerpt
- Time posted (relative: "2 saat önce", "dün", etc.)
- Favorite button (heart icon)

Click any card to open the full listing detail.

---

## Listing Detail

Full listing page includes:
- Full title and description
- Category, faculty, and posting date
- Contact info or notes from the listing owner
- Expiry date (if set by the owner)
- Favorites toggle
- Comments and Q&A section: any logged-in student can post a question or comment; the listing owner can reply (replies are indented under the original comment)
- Delete button (visible only to the listing owner)

---

## Posting a Listing ("İlan Ver")

1. Click the "İlan Ver" button in the top navigation
2. Fill in: Title (required), Description (required), Category (required), Faculty (optional), Image (optional), Expiry date (optional)
3. Click Publish — listing goes live immediately

---

## Favorites

- Click the heart icon on any listing to save it
- Access saved listings from the "Favoriler" tab in the navigation
- Unfavorite by clicking the heart icon again

---

## My Listings

- Access from the "İlanlarım" tab
- See all listings you have posted
- Close (deactivate) or delete any of your listings

---

## Comments and Q&A System

- Any logged-in student can comment or ask a question on any listing
- Comments appear in chronological order
- The listing owner has a special "İlan Sahibi" (Listing Owner) badge next to their name
- Only the listing owner can reply to top-level comments (replies appear indented below)
- The listing owner and comment author can both delete their own comments
